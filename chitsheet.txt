git status for check the status 
git add <fistatle> to add file in stagging 
git add -A to add all the file in stagging area at once
git commit for commit the file 
i to enable edit mode in vm editor
esc + : + w + q for quit from vm editor / git commit -m "added more htmls" shortcut for quit from vm editor
touch <filename> to create a new file
git checkout for one match(to replace working directory data to commited directory data)
git checkout -f for all match
git log to track commit 
git log -p <-(number of commit) to see specific number of commit
git diff to get difference between working directory and staging area directory
git diff --stage to compare last commit to stagging area
git commit - a -m to commit directory directly 
git rm <filename> to delete the file completly 
git rm --chached <filename> remove file from stagging area only
git status -s shows the current status of files
